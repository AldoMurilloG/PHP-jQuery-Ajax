Ejercicio 4
1) const jsonString = '{"nombre":"pepe","edad":30}';

const obj = JSON.parse(jsonString);


`jsonString` es una cadena de caracteres que representa un objeto en formato JSON.

`JSON.parse(jsonString)` convierte la cadena JSON en un objeto JavaScript.

Parámetros

El texto: jsonString (que es analizado por el método JSON.parse), se convertirá a JSON.

Returns

Retorna el objeto que se corresponde con el texto JSON entregado.

Exepciones

Lanza una excepción SyntaxError si la cadena a transformar no es un JSON válido.

Resolución:

El resultado mostrado será:

console.log(obj.nombre); // Imprime "pepe"

console.log(obj.edad); // Imprime 30


2) const jsonString = '{"nombre":"John","edad":30}';

const obj = $.parseJSON(jsonString);


El código para analizar es una cadena JSON en un objeto JavaScript. Utiliza jQuery en el método $.parseJSON.

Parámetros

$.parseJSON ( jQuery.parseJSON() ). Descripción: toma una cadena JSON bien formada y devuelve el valor de JavaScript resultante. Devuelve: Cadena o Número o Objeto o Matriz o Versión booleana en desuso: 3.0

Exepciones

El estándar JSON no permite "caracteres de control" como una pestaña o una nueva línea. Un ejemplo como $.parseJSON( '{ "testing":"1\t2\n3" }' ) arrojará un error en la mayoría de las implementaciones porque el analizador de JavaScript convierte la pestaña de la cadena y los escapes de nueva línea en una pestaña literal y una nueva línea; duplicar las barras diagonales inversas como "1\\t2\\n3" produce los resultados esperados. Este problema se ve a menudo al inyectar JSON en un archivo JavaScript desde un lenguaje del lado del servidor como PHP.

Hoy en día está en desuso, jQuery usa para analizar la cadena el método JSON.parse para obtener detalles sobre el formato JSON.


Ejercicio 5)
$json = '{"nombre":"pepe","edad":30}';

$data = json_decode($json, $data = true);


La función json_decode() se usa para decodificar o convertir un objeto JSON en un en una matriz u objeto asociativo de PHP.

Parámetros

json: El string de json a decodificar.

Esta función sólo trabaja con string codificados en UTF-8.

Valores devueltos

Devuelve el valor codificado en json en un tipo de PHP apropiado. Los valores true, false y null son devueltos como true, false y null respectivamente. null es devuelto si el parámetro json no se puede decodificar o si los datos codificados son más profundos que el límite de recursividad.
